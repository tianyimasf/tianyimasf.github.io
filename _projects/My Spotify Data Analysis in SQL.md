Top Played Songs at Different Times of Day (night / late night)

This project is a stream history analysis using MS SQL, Azure Data Studio, and Power BI.  
[View SQL Code in Github](https://github.com/tianyimasf/spotify-data-analysis)

- Data collection using OAuth2.0 and Spotify Web API using Python.
- Data cleaning, preparing, and transforming using Sheets and SQL.
- Extracted times played, days played, track and artist popularity, times in day played, and genre distribution for top frequently played tracks.
- Extracted top 20 played tracks in different times of day, for example, in the mornings or nights.
- Plug in distribution data in Power BI and created reports using histograms based on data.
